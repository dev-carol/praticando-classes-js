## Sobre classes

- Recurso inserido na versão ES6 do JavaScript
- O que simula as classes em outras linguagens orientadas a objetos
- Mas que por baixo dos panos, as classes se transformam em funções normais
- O que ajuda a escrever menos código, chega em um resultado mais rápido
- Traz recursos a ele de herança de uma forma mais fácil
- Tornando este método um syntatitc sugar
